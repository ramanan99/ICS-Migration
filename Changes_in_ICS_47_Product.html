If there's a dilemma (say a particular file is not covered in the below list), keep the following in mind:

If there's a difference and the change is only addition in 5.2 SP1, then have the 5.2 SP1 file in place.
If there're entries which are changed/removed in 5.2 product (compared to 4.7 SP4 ICS product), merge them
